# materialak

Hezkuntza-materialak antolatzeko biltegia.

## Egitura

- `docs/`: argitaratzeko prest dagoen edukia. GitHub Pages hemendik zerbitzatuko da.
- `docs/templates/`: material berriak sortzeko HTML txantiloiak.
- `docs/assets/`: estilo partekatuak eta irudiak.
- `src/`: lanerako oharrak, zirriborroak eta argitaratu gabeko materiala.

## Argitaratzea

Biltegia GitHub Pages bidez argitaratzeko prestatuta dago, `main` adarretik eta `/docs` karpetatik.
